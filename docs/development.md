# Development

## Commands

- `npm run format` — format all files with Prettier
- `npm run lint` — Prettier check + HTML validation + CSS lint
- `npm test` — alias to `npm run lint`
- `npm run build` — sanity check for required files

## GitHub Actions

- `CI` runs on PRs and pushes to `main`.
- `Pages` deploys the site on pushes to `main`.
