# Security Policy

## Supported Versions

This project is a static site (HTML/CSS). Security updates are provided for the latest version on the `main` branch.

## Reporting a Vulnerability

If you believe you found a security issue (e.g., unsafe links, sensitive data exposure, or supply-chain issues in tooling):

1. **Do not open a public issue** with exploit details.
2. Send an email to: **sousa3086@outlook.com**
3. Include:
   - A clear description of the issue
   - Steps to reproduce (if applicable)
   - Impact assessment
   - Suggested fix (optional)

You will receive an acknowledgement within **7 days**.
